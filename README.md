# Sintautai website template

A responsive HTML/CSS/JS homepage template inspired by the uploaded church sketch and coat of arms.

## Files

- `index.html` — homepage template
- `styles.css` — responsive layout and brand styling
- `script.js` — mobile navigation toggle
- `assets/church-sketch-hero.webp` — optimized hero background from the sketch
- `assets/church-sketch-hero.jpg` — fallback JPEG
- `assets/church-sketch-original.png` — original uploaded sketch
- `assets/coat-of-arms.png` and `assets/coat-of-arms.gif` — coat of arms
- `assets/clover.svg`, `assets/branch-divider.svg`, `assets/favicon.svg` — decorative brand elements
- `assets/news-*.svg`, `assets/town-river.svg` — replaceable placeholder images

## Usage

Open `index.html` in a browser, or place the folder on any static web server.

## Customization

Edit town name, contact details, news, events, and navigation directly in `index.html`.
Brand colors are CSS variables at the top of `styles.css`.
